#github基本操作命令
##从零开始创建项目远程仓库
- 在github页面上新建项目仓库
- 将github上的项目仓库克隆到本地

```
git clone https://github.com/Accumulations/github-operation-demo
```
- 将项目里所有文件添加到本地仓库`git add .`，并将文件上传到本地仓库且进行描述`git commit -m "github test"`
- 执行推送命令，将本地仓库推送到远程仓库`git push origin master`
